What you get

/hipaa-assess

20-minute organizational interview covering Security Rule, Privacy Rule, Breach Notification, and BAA requirements. Produces a gap analysis with prioritized findings.

/hipaa-scan

Automated scanning of your codebase (19 checks), cloud infrastructure (AWS/GCP/Azure), and IaC (Checkov + Rego). Maps every finding to a specific HIPAA requirement.

/hipaa-remediate

AI generates code patches, Terraform fixes, and 8 customized policy documents. SHA-256 hashed evidence for everything it produces.

/hipaa-report

Full compliance report, executive summary, and shareable trust report. Every requirement mapped, every finding linked to evidence.

Visual Dashboard

Real-time compliance dashboard at localhost:3000. Checklist, findings, risk register, vendor tracker, evidence library, exportable reports.

Ongoing Monitoring

Drift detection, vendor/BAA tracking, NIST risk assessment, and guided breach notification. Compliance is a process, not a checkbox.

Get started in 30 seconds

Requires Claude Code and Bun. Works with or without AWS/GCP/Azure credentials.

cd ~/.claude/skills && git clone https://github.com/aanishs/em-dash.git && cd em-dash && ./setup
